Book sample
--------------------

 Create, send and receive simple SOAP request

Steps to run the sample :
------------------------------------------

1. Start tomcat server

```shell script
$CATALINA_HOME/bin/startup.sh
```

2. Deploy sample

```shell script
mvn tomcat7:deploy
```

2. Open in browser following url and follow instructions

    http://localhost:8080/saaj-book
